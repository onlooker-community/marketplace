# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the Echo plugin. ADRs document significant design choices — the context, the options considered, and the rationale for the decision taken.

## Index

| ADR | Title | Status |
|-----|-------|--------|
| [0001](0001-tribunal-as-evaluator.md) | Use Tribunal subagents as the evaluation engine | Accepted |
| [0002](0002-score-delta-not-string-compare.md) | Compare rubric criterion scores, not output strings | Accepted |
| [0003](0003-baseline-as-committed-artifact.md) | Store baselines as committed JSON files in version control | Accepted |

## Format

ADRs use the MADR (Markdown Any Decision Record) format.
